# Early-Prediction-of-Student-Dropout-Risk-in-Online-Learning-Using-Machine-Learning
The rapid expansion of online education platforms has democratized access to learning, yet it has simultaneously introduced a persistent challenge: high student dropout rates. Many students enroll in courses but disengage before completion, often without any early warning system in place. This study presents a machine learning-based approach for the early prediction of student dropout risk using the Open University Learning Analytics Dataset (OULAD). The dataset encompasses student demographic profiles, virtual learning environment (VLE) interaction logs, and assessment performance records across multiple course presentations.

The original multi-class outcome variable - comprising Pass, Distinction, Fail, and Withdrawn - is transformed into a binary classification target. Students who fail or withdraw are labeled as at-risk (1), while those who pass or receive a distinction are labeled as not at-risk (0). Three data sources are merged to construct a unified feature set that captures student engagement, academic performance, and background characteristics. Features are engineered to address multicollinearity using Variance Inflation Factor (VIF) analysis, and separate preprocessing pipelines are built for Logistic Regression and XGBoost to align with each model's assumptions.

Experimental results demonstrate that XGBoost outperforms Logistic Regression across all evaluation metrics including accuracy, precision, recall, F1-score, and ROC-AUC. The findings confirm that VLE interaction features - particularly total clicks and active days - are strong predictors of dropout risk. This research demonstrates the practical potential of machine learning in enabling timely academic interventions, ultimately supporting improved student retention in online education systems.
